### 👋 About me

I’m a **full-stack** developer based in France 🇫🇷, and I build web applications from start to finish: robust back-end using **Symfony/PHP**, polished front-end in **JS/TS**, with particular attention paid to **CSS/SCSS**.

Aside from business logic, I spend quite a bit of time tinkering with my **Arch Linux + Hyprland** environment: dotfiles, installation scripts, dynamic theming… I like my tools to reflect my own style.

- 🔭 Currently working on a system of dotfiles and automated theming for Hyprland
- 🌱 Always learning and optimising my workflows
- ⚡ Fan of open-source tools
- 💬 Ask me about: Symfony, back-end architecture, or why Hyprland > everything else
